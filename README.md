## Instalasi

prequisite :
React Native
Expo

### 1. Clone repository

  $ git clone https://github.com/silvester-kw/modified-rc4-mobile

### 2. Install dependency

  $ npm install

### 3. Run development
  
  $ npx expo start

### 4. Buka melalui expo go atau emulator
